Legends of Dota Redux is built on the original Legends of Dota mod by ash47, with Redux developed and maintained by a team of contributors. Their roles are tracked in both the [README](https://github.com/DarkoniusXNG/Legends-of-Dota-Redux/blob/master/README.md) and the in-game `contributors.kv` file, which also powers the in-game credits screen and gold name feature.

## Emoji key

The README uses emoji to denote each contributor's role(s):

| Emoji | Role |
| :---: | ---- |
| 🌟 | Original Creator of LoD |
| 😈 | Redux Director |
| 👑 | Developer |
| 🍭 | Custom Ability Creator |
| 🌴 | Hero Perk Creator |
| 🚀 | Panorama |
| 😜 | Memes Redux Creator |
| 🚩 | Game Logic |
| ✨ | Developer of Other Mod |
| 🈷 | Chinese Translator |
| 🚣 | Creator of Overflow Abilities |

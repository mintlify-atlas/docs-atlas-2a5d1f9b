title: OP Abilities

The **OP (Overpowered)** ability pool is a separate collection of abilities that are too strong for regular play. They are hidden by default and only become selectable when the OP ability option is turned on in the game lobby.

## What makes an ability OP?

OP abilities typically have one or more of the following traits:

- Massively amplified versions of standard abilities (e.g., `dragon_knight_dragon_blood_OP` vs. `dragon_knight_dragon_blood`).
- Abilities with no theoretical stat cap that would grow without limit in a normal game.
- Mechanics that are fun in a single-player or sandbox context but break multiplayer balance.
- Standard abilities that Valve deemed too powerful for the hero rotation, now available in an explicitly unrestricted bracket.

## How to enable OP abilities

In the game lobby, open the **Advanced Options** panel and enable the **OP Abilities** toggle. Once enabled, all OP sub-brackets become visible in the skill picker alongside the normal pool.

## OP ability categories

OP abilities are organised into several sub-groups inside the `superop` bracket:

| Sub-group | Contents |
|---|---|
| `singlePlayer_actives` | Overpowered actives — Berserker's Call OP, Rupture OP, Dark Rift OP, and similar |
| `singlePlayer_passive` | Overpowered passives — Dragon Blood OP, Goblins Greed OP, Reactive Armor OP, Essence Aura OP variants |
| `singlePlayer_passive_2` | Additional OP passives — Aether Range OP, Octarine Vampirism OP, Angel Arena Razor Power OP |
| `singlePlayer_passive_3` | CD-reduction OP abilities — Jingtong OP, Zhanhun OP, Master Magic OP, Metamorphosis OP |
| `singlePlayer_passive_4` | Unique OP passives — Jinada OP, Warpath OP, Psi Blades OP, Atrophy Aura OP, Meepo Divided We Stand |
| `basic_abilities` (OP) | Stronger versions of stat bonus abilities — Mana Bonus OP, Health Bonus OP, Spell Amp OP, etc. |
| `survivor_op` | OP Survivor stack abilities |
| `spell_lab_souls_op` | OP Souls abilities |
| `memes_reborn_op` | OP fruit abilities — Banana, Blazer, Coconut, Grapefruit |
| `mega_ultimates` | Mega-tier ultimates — see below |

## Mega ultimates

Mega ultimates are extreme single-player abilities with map-wide or game-ending effects:

| Ability | Effect |
|---|---|
| Mega Black Hole | A vastly enlarged Black Hole that pulls in all units across a huge area |
| Mega Chronosphere | Freezes the entire map |
| Mega Eclipse | Global Eclipse strike coverage |
| Mega Freezing Field | Enormous Freezing Field radius |
| Mega Omni Slash | Multi-target Omnislash |
| Floor is Lava | Continuous AoE ground damage across the map |
| Mega Call Down | Global Gyrocopter missile barrage |
| Mega Flare | Global Flare damage |
| Mega Suicide | ... |

## Restrictions still apply

Even within the OP pool, troll combo rules remain active. Several OP abilities carry `ReduxBans` flags that prevent them from being picked alongside their normal counterparts (e.g., `aether_range_lod_OP` bans `aether_range_lod`). Cooldown reduction OP abilities are also part of the cooldown-reduction banned group.
